## Documentation home

This site is the published docs layer for JIGGAI products.

The long-term system is:

- product docs stay canonical in their home repos
- Mintlify presents those docs in one place
- synced pages are generated into this repo instead of manually copy-pasted

## Current state

This repo is being reshaped from the Mintlify starter into a product docs hub.

For now:

- `ClawRecipes/docs/` is the strongest existing canonical source
- `clawkitchen/docs/` is smaller and will grow over time
- `JIGGAIClawDocs` should remain the presentation layer, not a second manual source of truth
